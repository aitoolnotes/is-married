## 💡 Key Concepts

### How Posts Update Automatically

1. You edit `posts.json`
2. You commit/push to GitHub
3. Cloudflare notices the change
4. Site automatically rebuilds (30 sec)
5. Your post appears on homepage!

**No manual steps, no restart needed!**

### File Structure

```
GitHub Repo
    ├── index.html (homepage)
    ├── style.css (styling)
    ├── script.js (functionality)
    ├── posts.json (your posts)
    ├── sitemap.xml (SEO)
    └── robots.txt (SEO)
        ↓
    Cloudflare Pages
        ↓
    https://yoursite.pages.dev/
```
